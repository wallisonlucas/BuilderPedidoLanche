## Builder em um cenário real - Pedido de Lanche Personalizado

Projeto em Java que demonstra o uso do método **Builder** para criar pedidos personalizados em uma hamburgueria/pizzaria.

O sistema permite montar pedidos com diferentes opções, como tipo (hambúrguer ou pizza), tamanho, itens, forma de pagamento e entrega, garantindo que todas as regras de validação sejam aplicadas no método **Builder**.

O projeto inclui exemplos no Main com pedidos válidos e inválidos, demonstrando o funcionamento das validações e o uso do Builder.
